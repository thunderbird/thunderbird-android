# Milestone Issues

GitHub Milestone Issues define public delivery outcomes. They are created and managed by core maintainers because they
depend on GitHub permissions, project metadata, and roadmap coordination.

Use the [GitHub Milestone Issue template](0000-milestone-issue-template.md) when creating a new GitHub Milestone Issue.

GitHub Feature Issues, GitHub Task Issues, user journeys, RFCs, ADRs, and technical designs should link back to the
relevant GitHub Milestone Issue when they belong to that delivery outcome.

If work does not match a dedicated GitHub Milestone Issue, maintainers should link it to the relevant quarterly
catch-all GitHub Milestone Issue for community contributions or Android foundations work.
